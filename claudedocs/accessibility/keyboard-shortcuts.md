# PubHub Keyboard Shortcuts Guide

Complete reference for keyboard navigation and shortcuts in PubHub.

## Global Shortcuts

These shortcuts work from anywhere in the application:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘K` or `Ctrl+K` | Quick Navigation | Open command palette for fast navigation |
| `⌘⇧K` or `Ctrl+Shift+K` | Command Palette | Alternative command palette trigger |
| `Esc` | Close/Cancel | Close modals, dialogs, and command palette |
| `?` | Help | Show keyboard shortcuts help (when not typing) |

## Accessibility Features

### Screen Reader Support

PubHub is designed to work with screen readers:

- **NVDA** (Windows): All shortcuts work with NVDA
- **JAWS** (Windows): Compatible with JAWS navigation
- **VoiceOver** (macOS): Full VoiceOver support
- **TalkBack** (Android): Mobile touch exploration

### Keyboard-Only Navigation

PubHub is fully navigable without a mouse:

1. **Tab Order**: Logical tab order through all pages
2. **Focus Indicators**: Visible focus on all interactive elements
3. **No Keyboard Traps**: Always able to move away from elements
4. **Skip Links**: Skip to main content on page load

### Focus Management

- Focus automatically moves to opened modals
- Focus returns to trigger element when closing modals
- Focus trapped within modals (can't tab outside)
- Skip link available as first focusable element

## Tips for Keyboard Users

### Efficient Navigation

1. **Use Command Palette**: `⌘K` is the fastest way to navigate
2. **Learn Section Shortcuts**: Memorize main section shortcuts (⌘H, ⌘N, etc.)
3. **Tab Through Forms**: Use Tab to move through form fields efficiently
4. **Use Enter vs Space**: Enter activates, Space toggles/opens

### When Shortcuts Don't Work

Shortcuts are disabled when:
- Typing in text inputs, textareas
- Editing content in rich text editors
- Focused on search fields

To re-enable shortcuts, press `Esc` or tab away from the input.

## Browser Shortcuts

Some shortcuts may conflict with browser shortcuts:

| PubHub Shortcut | Browser Alternative | Solution |
|-----------------|---------------------|----------|
| `⌘N` | New Window | Use command palette instead |
| `⌘T` | New Tab | Disabled when typing, or use ⌘K |
| `⌘W` | Close Tab | Not used by PubHub |

## Mobile Touch Gestures

While keyboard shortcuts don't apply to mobile, these gestures work:

| Gesture | Action | Description |
|---------|--------|-------------|
| Swipe Right | Back | Navigate back (where applicable) |
| Pull Down | Refresh | Refresh current view |
| Long Press | Context Menu | Show context menu for items |
| Double Tap | Activate | Activate elements with TalkBack/VoiceOver |

## Troubleshooting

### Shortcuts Not Working?

1. **Check Focus**: Ensure you're not typing in an input field
2. **Press Esc**: Exit current context and try again
3. **Reload Page**: Some shortcuts require page reload after changes
4. **Check Browser**: Some browsers may override shortcuts

### Report Keyboard Issues

If you encounter keyboard navigation problems:
- Email: accessibility@pubhub.dev
- Include: Browser, OS, specific shortcut, expected vs actual behavior
