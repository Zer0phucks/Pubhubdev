# OAuth Callback URLs - Registration Guide

## Callback URLs Needed for Each Platform

Add these callback URLs in each platform's developer console:

| Platform | Callback URL to Register |
|----------|-------------------------|
| Twitter | ✅ Already has: `https://pubhub.dev/api/oauth/callback/twitter` |
| Instagram | `https://pubhub.dev/api/oauth/callback/instagram` |
| Facebook | `https://pubhub.dev/api/oauth/callback/facebook` |
| LinkedIn | `https://pubhub.dev/api/oauth/callback/linkedin` |
| YouTube | `https://pubhub.dev/api/oauth/callback/youtube` |
| TikTok | `https://pubhub.dev/api/oauth/callback/tiktok` |
| Pinterest | `https://pubhub.dev/api/oauth/callback/pinterest` |
| Reddit | `https://pubhub.dev/api/oauth/callback/reddit` |
