## Code Example

### Component Usage

```tsx
import { AppHeader } from "./components/AppHeader";

function App() {
  const [selectedPlatform, setSelectedPlatform] = useState<Platform>("all");
  
  return (
    <AppHeader
      currentView="home"
      selectedPlatform={selectedPlatform}
      onPlatformChange={setSelectedPlatform}
      onNavigate={(view, subView) => {
        // Handle navigation
        if (view === "settings" && subView === "connections") {
          // User clicked "+ Connect Platform"
        }
      }}
      // ... other props
    />
  );
}
```

### Hook Usage

```tsx
import { useConnectedPlatforms } from "./components/useConnectedPlatforms";

function MyComponent() {
  const { connectedPlatforms, hasUnconnectedPlatforms, refresh } = useConnectedPlatforms();
  
  console.log(connectedPlatforms); 
  // Output: ["twitter", "instagram", "linkedin"]
  
  console.log(hasUnconnectedPlatforms);
  // Output: true (because 3 out of 9 platforms are connected)
  
  return (
    <div>
      <p>Connected: {connectedPlatforms.length} / 9 platforms</p>
      {hasUnconnectedPlatforms && (
        <button onClick={() => navigate('/settings/connections')}>
          Connect More Platforms
        </button>
      )}
    </div>
  );
}
```

## Benefits Summary

| Aspect | Before | After |
|--------|--------|-------|
| **Clutter** | Shows all 9 platforms | Shows only connected |
| **Guidance** | No clear next action | "+ Connect Platform" CTA |
| **Empty State** | Shows all grayed out | Clear "No platforms" message |
| **Scalability** | Fixed 9 tabs always | Grows with connections |
| **Mobile** | Requires lots of scrolling | Compact, relevant tabs |
| **Project-aware** | Static | Dynamic per project |
